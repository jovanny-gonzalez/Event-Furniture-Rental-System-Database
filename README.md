# Event-Furniture-Rental-System-Database
The Event Furniture Rental System Database was a team project made Daira Aguilar, Alonso Benoun, Isaac Esquivel, Jovanny Gonzalez(me), and Karina Garcia-Carrillo(Leader). It's purpose it to manage furniture reservations for events such as weddings, parties, and corporate gatherings.

# Role: Database Design Lead
-Designed the ER Diagram and used MySQL forward engineering to generate the relational schema.
-Modeled one-to-many and many-to-many relationships using primary keys, foreign keys, and a bridge table.
-Designed table structures for customers, events, bookings, furniture inventory, and booking-to-furniture bridge records.
